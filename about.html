ABOUT

关于这个项目

一个写在桌面角落的小时钟, 后来长成了完整的世界。

项目由来

灵境时钟最初只是想把"时间"这件事做得更安静 — 桌面上的时钟, 不必喧哗, 像一面镜子, 映出当下的模样。

慢慢加入了天气、农历、每日一言、Live2D 角色。每加一点, 都在思考一件事: 它该以什么方式存在? 于是有了"灵境UI" 和"灵镜光感" — 把所有视觉决策, 收束到一套设计语言里。

屏幕如镜, 光从内生 — 这是灵境时钟对自己的全部要求。

核心理念

克制, 不喧哗 — UI 高频用字保持易识别, 营销层克制。漂亮不等于花哨, 留白本身就是表达。

离线, 不打扰 — 定位优先使用本地浏览器能力, 网络只在必要时介入。无广告、无追踪、无后台服务。

开源, 可审 — 代码公开、API key 加密 (Base64 + 拆分, 仅为防止明文暴露, 核心价值在密钥本身), 任何人都可以打包验证。

DESIGN LANGUAGE

灵境UI / 灵镜光感

自研设计语言 — 把每一处光, 都归到四个象限。

01
毛玻璃
Frosted Glass

backdrop-filter: blur(40px) saturate(1.8) — 透过磨砂看见世界, 信息保持可读, 视觉保持通透。

02
粒子背景
Particle Field

Canvas 2D 渲染的微光场, 鼠标经过时粒子自动让路, 静中有动。

03
模糊发光
Soft Glow

box-shadow 与 filter:blur 配合, 营造有呼吸感的边缘光晕, 让界面不显得"硬"。

04
暗角
Vignette

径向渐变覆盖层, 把目光引向中心, 同时降低边缘的视觉权重。

隐私说明

灵境时钟不收集任何用户数据, 不发送任何遥测, 无后台服务进程。天气查询时, 仅在获得你授权后访问 api.open-meteo.com 与和风天气 API, 用于获取当前位置天气。

API key 在代码中经过 Base64 + 拆分混淆, 仅为防止搜索引擎抓取明文泄露额度 — 任何人都可以从代码中恢复, 但若你打算 fork 后分发, 请替换为你自己的和风天气 key。
