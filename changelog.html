v1.2.0 2026 · 06 · 09 新主功能

灵境 安装仪式 + 自动更新 — 让升级也变得轻巧。

  • 🎁 灵境 安装仪式 (Splash) — 复制文件完成后, 灵境时钟自身弹一个 480×360 居中圆角毛玻璃 splash 仪式页: 1.4s 冷白月光光毯 + 灵字 Logo 浮呼吸 + 紫青极光进度条 + 4.5s 自动退场
  • 🖼️ NSIS 安装器品牌化 — 头图 164×55 + 侧栏 164×314 + 安装器图标 256×256 + 头图图标 32×32 全部烧灵境品牌色 (紫青极光渐变 + 灵字 Logo)
  • ⚙️ 安装选项页 (nsDialogs 自绘) — 路径选择后, 安装前弹一个"安装选项"页, 3 个勾选: 启用自动检查更新 (推荐, 默认勾上) + 创建桌面快捷方式 + 创建开始菜单快捷方式; 旁注明确告知"需要联网获取新版本, 您随时可在 设置 中关闭"
  • 🔔 自动检查更新 (electron-updater) — 启动 6s 后静默 checkForUpdates(), 有更新后台下载, 装好弹非阻塞 toast (右上角毛玻璃卡片); 不强制重启, 用户点"立即重启"才生效
  • 📜 精简用户协议 (LICENSE.txt) — ~200 字, 4 条 (自由使用 / 不保证质量 / 数据归属 / 反馈与贡献), 不堆诱导分享 / 争议仲裁 / 冗长免责条款
  • 🪟 自定义 Welcome / Finish / Unwelcome 页 — 灵境色头图 + 标题, 装完引导"立即体验灵境时钟"
  • 设置 → 系统更新 — "自动检查更新"开关 + "立即检查更新"按钮, 装时勾选值同步到 app
v1.1.0 2026 · 06 · 09 新主功能

灵境 Onboarding — 让首次启动也有仪式感。

  • 🌅 首次启动毛玻璃欢迎页 — 应用启动时 (仅一次) 弹出全屏毛玻璃引导, 1.4s 冷白月光光毯横扫入场, 3 张引导卡 (右下角热区 / 设置面板 / 快捷键) 错峰淡入
  • ✨ "重新显示欢迎页" 入口 — 设置面板底部加链接, 一键重看 Onboarding
  • 💫 主按钮点击 ripple + 卡片 hover glow + 主按钮呼吸动效
  • 可访问性 — role="dialog" aria-modal="true", 焦点自动跳到主按钮, 关闭时还原; prefers-reduced-motion 兜底
  • 键盘 — Esc 关闭 (等同"稍后再说") / Enter 触发"开始使用"
  • 异常兜底 — localStorage 写失败不阻断应用, 异常时序下 flag 仍 atomic 写
v1.0.1 2026 · 06 · 08 增强

这次让桌面时钟终于"听你的话"。

  • 设置持久化到 localStorage, 启动自动恢复
